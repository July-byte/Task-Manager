# 📝 Task Manager (Console App)

Simple console task manager written in C# (.NET).

This project was created as part of my learning journey in C# and .NET development.

---

## 🚀 Features

- ➕ Add new tasks
- 📋 View all tasks
- ✅ Mark task as completed
- ❌ Delete tasks
- 🔍 Filter completed tasks
- 💾 Automatic save to file
- 📂 Load tasks on startup

---

## 🛠 Technologies Used

- C#
- .NET 8
- Object-Oriented Programming (OOP)
- LINQ
- File I/O
